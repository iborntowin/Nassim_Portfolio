# InventoryERP Media Assets (Portfolio)

This folder belongs to the portfolio website repository and stores screenshots used by the InventoryERP case study.

It is not the full InventoryERP source code repository.

## What This Folder Is For

- Image assets rendered in the project details page and showcase cards.
- Static media consumed by Next.js from `public/images/projects/erp/*`.

## Related Portfolio Data Sources

- `src/lib/projects-data.ts` contains InventoryERP metadata (ID `7`), stack, stats, and image references.
- `src/components/pages/erp-detail-page.tsx` renders the ERP project page.
